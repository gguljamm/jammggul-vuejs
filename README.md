# JammGgul

> Create a personal homepage using vue3

## Objective

``` bash
# 재미있는 홈페이지.
vue3를 최대한 활용하여 만들어보기.

# firebase 활용
firestore, storage, hosting 등 적용.

# firebase DB 활용해서 게시판
이미지, 글, 날짜 등의 정보 담은 게시판 만들기
```
